# Домашние задания по курсу «Программирование на Python»

Данный репозиторий содержит выполненные домашние задания в рамках курса **«Программирование на Python»**.

Каждое задание оформлено в виде отдельного скрипта или набора файлов, размещенных в отдельной папке в репозитории, и сопровождается кратким описанием и инструкцией по запуску.

Каждое задание размещается в отдельной ветке, которая далее сливается в master.
